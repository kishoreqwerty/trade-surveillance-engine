# calibration/

Offline calibration of `simulator/`'s synthetic baseline parameters against WRDS/TAQ stylized facts (volume clustering, spread behavior, cancellation rates). Phase 11 of the build guide.

This directory holds calibration **scripts and output parameter values only**.

**Never commit raw or derived WRDS/TAQ data to this repo, in any form, at any path.** TAQ access is licensed for offline academic research use, not redistribution. Confirm exact redistribution terms once WRDS access is registered, before adding anything here beyond scripts.
